# Derivadas Simbólicas

Este projeto implementa o cálculo de derivadas simbólicas usando a biblioteca `sympy`.

## Função Implementada

A função utilizada é:
\[
y = x^3
\]

## Método

- **Derivada Simbólica**: A derivada é calculada usando a função `diff` do `sympy`.
